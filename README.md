## Description of Adventure
For this adventure, I chose to contribute to and advance the development of **EzSqueeze**, an open-source professional-grade audio dynamics processor. My work involved implementing core Digital Signal Processing (DSP) algorithms, designing a modern "Citrus Liquid-Glass" user interface using the HISE/JUCE framework, and setting up a robust CI/CD pipeline. This project combines low-level C++ programming with high-level UI/UX design.

## Summary of Deliverable
The primary deliverable is a functional C++17 audio plugin (VST3/AU). Key components include:
* A **DSP Engine** featuring Peak/RMS detection, Lookahead, and Oversampling.
* A **Modern UI** built with the HISE framework, emphasizing accessibility and visual feedback.
* **Build Automation** via CMake and GitHub Actions for cross-platform compatibility (Windows/macOS).
* **Technical Documentation** covering architecture, DSP math, and build instructions.
